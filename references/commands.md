# 命令参考

| 命令 | 功能 | 示例 |
|------|------|------|
| `/duckdb 分析` | 执行数据分析 | `/duckdb 分析 最近 30 天的订单` |
| `/duckdb 问数` | 重用历史查询 | `/duckdb 问数 订单统计` |
| `/duckdb 运行预测` | 执行预测任务 | `/duckdb 运行预测 销售预测` |
| `/duckdb 记录` | 查看历史记录 | `/duckdb 记录 sql` |
| `/duckdb 配置` | 配置管理 | `/duckdb 配置 测试连接` |
| `/duckdb 帮助` | 查看帮助 | `/duckdb 帮助` |

---

## 命令详解

### /duckdb 分析

执行完整数据分析流程。

**参数**:
- 查询描述（必需）：自然语言描述

**示例**:
```
/duckdb 分析 最近 30 天的订单数据
```

**流程**:
1. 生成 SQL
2. 用户确认
3. 执行查询
4. 保存记录

---

### /duckdb 问数

重用历史查询记录。

**参数**:
- 记录 ID 或关键词（必需）

**示例**:
```bash
# 精确匹配
/duckdb 问数 query_20260319_140000_abc123

# 模糊匹配
/duckdb 问数 订单统计
```

---

### /duckdb 运行预测

执行预测任务。

**参数**:
- 预测任务 ID 或关键词（必需）

**示例**:
```bash
/duckdb 运行预测 pred_xxx
/duckdb 运行预测 销售预测
```

---

### /duckdb 记录

查看历史记录。

**参数**:
- 类型（可选）：`sql` / `预测` / `全部`（默认）
- 日期（可选）：`YYYY-MM-DD`（默认：今天）

**示例**:
```bash
/duckdb 记录
/duckdb 记录 sql
/duckdb 记录 预测 2026-03-19
```

---

### /duckdb 配置

配置管理命令。

**子命令**:
- `查看` - 查看当前配置
- `测试连接` - 测试 DuckDB 连接

**示例**:
```bash
/duckdb 配置 查看
/duckdb 配置 测试连接
```

---

### /duckdb 帮助

查看帮助信息。

**参数**:
- 主题（可选）：命令名称

**示例**:
```bash
/duckdb 帮助
/duckdb 帮助 分析
```

---

## 用户确认流程

### SQL 执行确认

```
助手：已生成以下 SQL：

SELECT ... FROM ...

请确认：
- "确认" → 执行查询
- "修改 XXX" → 调整 SQL
- "取消" → 取消
```

### 预测任务确认

```
助手：已设计预测方案：

- 预测目标：...
- 模型：...
- 执行频率：...

请确认：
- "确认" → 创建任务
- "修改 XXX" → 调整参数
- "取消" → 取消
```
